# Improved Kaleidoscope LLVM Example Code

The LLVM Kaleidoscope example is nice in principle, but its author
openly admits that it doesn't provide idiomatic code.  Moreover, I
find that it doesn't match LLVM's own style guide and does
capitalization in a way that I dislike.

This is some C++ code that implements the same tutorial content, but
with an eye towards being more idiomatic.  It also supports unicode
and some test files to run against.
